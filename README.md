# Controle de uma coluna de destilação coluna
Este projeto apresenta um sistema para o controle automático de uma coluna de destilação com um sistema supervisório e opções de controle e identificação utilizando metodos classicos e modernos. O sistema foi projetado para um coluna específica localizada no laboratório de enegnheria quimica da UEM. Entretanto pode ser adaptado para outras colunas. Esta aplicação é inserida em um raspberry que realiza a interação com os intrumentos de campo.

## Como funciona
Devo acessar o arquivo main.py que abrirá a tela principal do sistema, nestatela é possivel visualizar infomrações da coluna e atuar na mesma. Para um controle ou identificação da coluna é necessário acessar o menu lateral e escolher o mode de controle e identificação.
